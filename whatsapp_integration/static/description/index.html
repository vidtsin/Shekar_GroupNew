Integrate Odoo with WhatsApp

Send messages to your customers via WhatsApp messaging service
Fully Automated System.



Contact your WhatsApp contacts directly from Odoo

Easy way to send messages from Odoo to your customers via WhatsApp



Send message to multiple contacts with attachments

Send WhatsApp message to as many contacts as you want, thanks to modified many2many widget. Furthermore you can attach multiple attachments with the message.



Login to WhatsApp made easy

Just scan QR code from your mobile device and you are good to go

If you don't know how to login to WhatsApp, just follow the instruction from here
All users of odoo can login to their WhatsApp individually, i.e. each Odoo user will have their own Session. They can Logout from general settings. Also they have Logout option in their WhatsApp Mobile App.



Logout from WhatsApp made easy

There is an option in Configuration -> General Settings to Logout



Requirements

phonenumbers python library -
sudo pip3 install phonenumbers
selenium python library -
sudo pip3 install selenium

Install Chromium or Google Chrome browser (Skip if already installed - Just update it)
Chromium
sudo apt install --assume-yes chromium-browser
Google Chrome
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo dpkg -i google-chrome-stable_current_amd64.deb
If any error occurs during Google Chrome Installation
sudo apt-get install -f
